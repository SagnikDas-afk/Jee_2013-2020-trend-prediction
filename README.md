# Jee_2013-2020-trend-prediction

### JEE Exam Success Prediction using PyTorch. A deep learning project implementing a multi-layer perceptron to predict student pass/fail outcomes based on subject marks, featuring data visualization of training loss and performance trends.

### JEE Exam Performance Analysis & Prediction

### Framework: PyTorch (Neural Networks)

### Data Processing: Pandas, NumPy, Scikit-Learn
